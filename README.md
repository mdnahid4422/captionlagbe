# CaptionLagbe (Structured)

## Folder Structure
- index.html
- css/
  - base.css (current full UI styles)
  - layout.css, components.css, auth.css (placeholders for future split)
- js/
  - userStorage.js (localStorage-based auth)
  - auth.js (login/signup modal + logout fix)
  - data.js (captionsData)
  - utils.js, state.js, ui.js (for future modular growth)
  - app.js (main UI logic)

## Run
Just open `index.html` in your browser (or use VS Code Live Server).
